# Corrupt Heart
The final boss, found at the end of Act 4.
## Pattern
| Name         | Intent | Effect                                                                                              |
|--------------|--------|-----------------------------------------------------------------------------------------------------|
| Debilitate    |        | Applies 2 Vulnerable, 2 Weak and 2 Frail.Adds 1 Burn, 1 Dazed, 1 Slimed, 1 Void and 1 Wound into your draw pile. |
| Blood Shots  |        | Deals 2x12 damage.                                                                                  |
|              |        | Deals 2x15 damage.                                                                                  |
| Echo         |        | Deals 40 damage.                                                                                    |
|              |        | Deals 45 damage.                                                                                    |
| Buff         |        | Loses all negative Strength, then gains 2 Strength. Gains an additional Buff:                            |
|              |        | - 1st buff: 1 Artifact                                                                              |
|              |        | - 2nd buff: 1 Beat of Death                                                                          |
|              |        | - 3rd buff: 1 Painful Stabs                                                                         |
|              |        | - 4th buff: 10 Strength                                                                             |
|              |        | - 5th buff: 50 Strength                                                                             |
Always starts with Debilitate. Uses Buff every 3 turns starting from turn 4. On all other turns it has a 50/50 chance to use either Blood Shots followed by Echo or Echo followed by Blood Shots.
## Trivia
- Mental Fortress activates before Beat of Death for every Stance changing card, except for Indignation which activates after.
## Powers
- **Beat of Death (1)**: Whenever you play a card, take X damage.
- **Invincible**: Can only lose X more HP this turn.

| Bash    |        | Deals 12 damage. If the Player has an Orb Slot, there is 50% chance to apply -1 Focus. Otherwise applies -1 Strength. |

| Fortify |        | All enemies gain 30 Block.                                                                               |

|         |        | Deals 38 damage. Gains Block equal to damage dealt.                                                        |

Every three turns, starting from turn 3, it uses Smash. Otherwise has a 50/50 chance of using:
- Bash followed by Fortify
- Fortify followed by Bash
## Found With

# Spire Spear
Spire Spear is an Elite enemy found exclusively in Act 4.
## Pattern
| Name       | Intent | Effect                                                                 |
|------------|--------|------------------------------------------------------------------------|
| Burn Strike|        | Deals 5x2 damage. Adds 2 Burns into your discard pile.                    |
|            |        | Deals 6x2 damage. Adds 2 Burns into your discard pile.                    |
|            |        | Deals 6x2 damage. Adds 2 Burns on top of your draw pile.                   |
| Skewer     |        | Deals 10x3 damage.                                                     |
| Piercer    |        | Deals 10x4 damage.                                                     |
|            |        | All enemies gain 2 Strength.                                            |
Always starts with Burn Strike. Every three turns, starting from turn 2, it uses Skewer. Otherwise has a 50/50 chance of using:
- Burn Strike followed by Piercer
- Piercer followed by Burn Strike
## Found With
Act 4
- Spire Shield on the left side of the player.
## Notes
- The player begins this elite combat with the Debuff Surrounded. This only serves as a telegraph for their Back Attack mechanic and does not actually add any damage bonus.
- Smoke Bomb cannot be used while Surrounded.
## Powers
- **Artifact 1 (1)**: [Description not provided in the image]
- **Back Attack**: Deals 50% more damage as it is attacking you from behind.
